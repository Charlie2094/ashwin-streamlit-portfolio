# ⚡ Ashwin Gopal · Portfolio App

> A Streamlit portfolio app that **live-fetches** HTML files from GitHub — always up to date automatically.

---

## 🚀 How It Works

```
User opens Streamlit app
        ↓
app.py fetches latest HTML from GitHub (every 5 mins)
        ↓
Renders Portfolio / Architecture / Flow inline
        ↓
Always shows latest version ✅
```

---

```
└── README.md
```

---

## ⚙️ Deploy to Streamlit Cloud

1. Go to **share.streamlit.io**
2. Sign in with GitHub
3. Click **"New app"**
4. Select this repo
5. Set **Main file path** → `app.py`
6. Click **Deploy** ✅

---

## 🔄 Update Workflow

```
Edit any HTML file locally
        ↓
Upload to github.com/Charlie2094/Ashwin-portfolio/HTML/
        ↓
Streamlit auto-refreshes within 5 minutes ✅
No changes needed here!
```
